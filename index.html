Paso a paso - SASS

  1. Descargar e Instalar Node.js
  2. Entrar en la consola o terminal y posicionarse en la carpeta del proyecto donde queremos inicializar SASS.
  3. Ejecutar en orden los siguientes comandos:
    1. npm init
    2. Resonder todas las preguntas apretando enter y al final de todo cuando nos pregunte escribir "yes".
    3. npm install -D node-sass nodemon
  4. La carpeta css ya la tienen creada con su archivo adentro, por lo tanto tienen que crear una carpeta scss con un archivo style.scss
  5. Abrir package.json con vsc o sublime para poder editarlo.
  6. Pegar la linea de build-css y watch-css debajo de "test" (agregar una "," al final de esta linea).

    1. "build-css": "node-sass --include-path scss scss/style.scss css/estilo.css",
      "watch-css": "nodemon -e scss -x \"npm run build-css\""

  7. Verificar que los nombres de los archivos CSS y SCSS sean los correctos en la línea "build-css".
  8. Ejecutar el último comando y empezar a trabajar
    1. npm run watch-css
